# Atomic Command Center

Plataforma de gestión inteligente para múltiples cuentas de WhatsApp con análisis de IA (OpenAI).

## Visión
Optimizar el flujo de trabajo de los líderes de ventas permitiéndoles gestionar múltiples cuentas de WhatsApp desde un solo panel, con sugerencias automáticas generadas por IA para mejorar el cierre de ventas y el seguimiento de clientes.

## Características Principales
- **Gestión Multi-Cuenta**: Conecta varios números de WhatsApp simultáneamente.
- **Persistencia de Sesión**: Inicia sesión una vez y mantén la cuenta activa para todo el equipo.
- **Análisis con IA**: Procesamiento de mensajes en tiempo real para sugerir respuestas óptimas.
- **Panel Web**: Interfaz moderna para visualizar actividad y aprobar respuestas.

## Estructura del Proyecto
- `backend/`: Servidor Node.js con integración de `whatsapp-web.js` y OpenAI.
- `frontend/`: Aplicación React para el dashboard de administración.
- `docs/`: Documentación original y guías del proyecto.

## Requisitos
- Node.js (v18+)
- MongoDB
- API Key de OpenAI
